## 🧭 What zebra does

zebra helps you anonymize e-invoices on your own computer. It works with ZUGFeRD and XRechnung files in XML and hybrid PDF form. It keeps your data on your device, so you do not need to upload invoices to a server.

Use zebra when you want to remove personal data before sharing an invoice with a client, partner, auditor, or support team. It runs in your browser as a client-side app, which means the file stays on your PC while you work.

## 💻 What you need

zebra is made for Windows users who want a simple local app. A modern Windows system should be enough.

You should have:

- Windows 10 or Windows 11
- A recent web browser, such as Edge, Chrome, or Firefox
- Enough space to save and open invoice files
- Internet access only for the first download

If you open large PDF invoices, a system with more memory will help. For most normal invoice files, a standard home or office PC should work fine.
